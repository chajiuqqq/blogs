给blog添加Disqus评论功能

2023-08-07 12:55
阅读时长: 1 分钟

stack是本blog使用的hugo主题，今天研究了一下如何配置disqus功能。

stack本身支持多种comments系统，只要在config配置下即可。

可以从文档开始,然后看DisqusJS的介绍。

  1. 先在Disqus注册APP，然后在 Disqus API Application处注册一个 Application。

  2. 获取API Key ，添加domain。

  3. 在Advanced这里还要加个doamin。

  1. 然后主要修改config以下几个地方：
  • 根目录下的disqusShortname，从这里获取，这边文档给了WARNING，说是必须从这里修改，不然无法起作用，确实是这样。
  • comment下的disqusjs: apiUrl抄我这个就可以。apiKey和shortname用你自己的，不确定shortname这边还有没有必要填。
